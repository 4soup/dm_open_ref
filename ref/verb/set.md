# settings (verb)


verb settings:
    verb/set
      name
      desc
      category
      hidden
      popup_menu
      instant
      invisibility
      src
      background


Procs and verbs are the same \"type\" so these attributes may
be set for procs as well; most of them do not have any meaning, however,
unless the proc is invoked as a verb (by adding it to a verb list).
