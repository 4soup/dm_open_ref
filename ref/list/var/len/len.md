## len var (list)    
**See also:**    
:   [+ operator](/ref/operator/+.md)    
:   [+= operator](/ref/operator/+=.md)    
:   [- operator](/ref/operator/-.md)    
:   [-= operator](/ref/operator/-=.md)    
This is the length of the list. Increasing it expands the list,    
initializing all new elements with null. Decreasing it contracts the    
list, making old elements inaccessible.
