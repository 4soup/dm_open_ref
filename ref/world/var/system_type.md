## system_type var (world)
**See also:**
*   [byond_version var (world)](/ref/world/var/byond_version.md) -m
*   [process var (world)](/ref/world/var/process.md) -m
*   [shell proc](/ref/proc/shell.md) -m

This variable indicates the operating system type at run-time.
It will be one of the following constants:
-   MS_WINDOWS
-   UNIX
