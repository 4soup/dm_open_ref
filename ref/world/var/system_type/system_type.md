## system_type var (world)    
**See also:**    
:   [byond_version var (world)](ref/world/var/byond_version)    
:   [process var (world)](ref/world/var/process)    
:   [shell proc](ref/proc/shell)    
This variable indicates the operating system type at run-time. It will    
be one of the following constants:    
-   MS_WINDOWS    
-   UNIX
