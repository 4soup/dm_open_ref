## host var (world)
**See also:**
*   [game_state var (world)](/ref/world/var/game_state.md) -m
*   [name var (world)](/ref/world/var/name.md) -m
*   [status var (world)](/ref/world/var/status.md) -m
*   [visibility var (world)](/ref/world/var/visibility.md) -m<!-- -->
**Default value:**
*   null


If the information is made available by the pager, this will
provide the key of the world\'s host. If the host is not known, this
value will be either null or an empty string.
