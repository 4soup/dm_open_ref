## byond_build var (world) 
###### BYOND Version 512
**See also:**
+   [DM_VERSION macro](/ref/DM/preprocessor/DM_VERSION.md) 
+   [byond_version var (world)](/ref/world/var/byond_version.md) 
+   [byond_version var (client)](/ref/client/var/byond_version.md) 
+   [byond_build var (savefile)](/ref/savefile/var/byond_build.md) 
+   [byond_version var (savefile)](/ref/savefile/var/byond_version.md) 

This is the build number (minor version) of BYOND being run by
this server. Typically this is not useful information, but it can come
in handy when diagnosing issues reported by players when hosting with a
beta build.
