## key var (client)    
**See also:**    
:   [key var (mob)](/ref/mob/var/key/key.md)    
<!-- -->    
**Default value:**    
:   The player\'s key identity.    
This is a read-only value that contains the player\'s key. Once the    
player is attached to a mob M, M.key == M.client.key.
