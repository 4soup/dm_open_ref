*   [\_\_IMPLIED_TYPE\_\_ macro](/ref/DM/preprocessor/__IMPLIED_TYPE__.md) -m

The \_\_FILE\_\_ macro expands to a string containing the name
of the current source file. This may be useful when generating debugging
error messages.
