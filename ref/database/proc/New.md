## New proc (database) 
###### BYOND Version 506
**See also:**
*   [database datum](/ref/database.md) 
*   [Open proc (database)](/ref/database/proc/Open.md) <!-- -->
**Format:**
*   New(filename)
<!-- -->
**Args:**
*   filename* The database filename to open (optional)


Creates a new database datum, and opens the file if a filename
is provided.
