## can-scroll parameter (skin)    
**See also:**    
:   [on-size parameter](/ref/%7Bskin%7D/param/on-size/on-size.md)    
:   [size parameter](/ref/%7Bskin%7D/param/size/size.md)    
<!-- -->    
**Applies to:**    
:   [Main](/ref/%7Bskin%7D/control/main/main.md) (pane only)    
<!-- -->    
**Possible values:**    
:   none    
:   horizontal    
:   vertical    
:   both    
<!-- -->    
**Default value:**    
:   none    
Allow this pane to retain its horizontal and/or vertical size and show    
scrollbars if necessary, instead of shrinking to fit the container.
