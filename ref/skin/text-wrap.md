## text-wrap parameter (skin)

<!-- -->
**Applies to:**
+   [Label](/ref/%7Bskin%7D/control/label.md) <!-- -->
**Format:**
+   true/false
<!-- -->
**Default value:**
+   false


Wrap text that is too long for the width of the label.

> **See also:**
> +   [text parameter](/ref/%7Bskin%7D/param/text.md)
