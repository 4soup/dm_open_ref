## text-wrap parameter (skin)    
**See also:**    
:   [text parameter](/%7Bskin%7D/param/text)    
<!-- -->    
**Applies to:**    
:   [Label](/%7Bskin%7D/control/label)    
<!-- -->    
**Format:**    
:   true/false    
<!-- -->    
**Default value:**    
:   false    
Wrap text that is too long for the width of the label.
