**Applies to:**    
:   [Main](/ref/%7Bskin%7D/control/main.md) (window only)    
<!-- -->    
**Format:**    
:   true/false    
<!-- -->    
**Default value:**

Allow the window to be closed, and also shows a system menu for the    
window.
