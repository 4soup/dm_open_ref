## title parameter (skin)    
**See also:**    
:   [name var (world)](/ref/world/var/name/name.md)    
:   [icon parameter](/ref/%7Bskin%7D/param/icon/icon.md)    
<!-- -->    
**Applies to:**    
:   [Main](/ref/%7Bskin%7D/control/main/main.md)    
<!-- -->    
**Format:**    
:   string    
The title of this window or pane. For a window, the title will appear in    
the titlebar if present. For a pane, this will be displayed on the tab    
if this pane is in a [Tab control](/ref/%7Bskin%7D/control/tab/tab.md).    
If this is the default window, [world.name](/ref/world/var/name/name.md){.code}    
takes precedence over the window title.
