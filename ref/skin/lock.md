**Applies to:**    
:   [Child](/%7Bskin%7D/control/child)    
<!-- -->    
**Possible values:**    
:   left    
:   right    
:   none    
<!-- -->    
**Default value:**    
:   none    
Allows one pane to \"lock\" the splitter so if this Child control is    
resized, the splitter will stay put on that side.
