## border parameter (skin)    
**Applies to:**    
:   [All]/%7Bskin%7D/control    
<!-- -->    
**Possible values:**    
:   none    
:   line    
:   sunken    
<!-- -->    
**Default value:**    
:   none    
Border type around the control or window. May not work the same in all    
controls.
