## screen-size parameter (skin) 
###### BYOND Version 515

<!-- -->
**Applies to:**
+   [Main](/ref/skin/control/main.md) <!-- -->
**Format:**
+   *width*x*height*


Read-only. 

For windows, this is the size of the
nearest monitor\'s area.
