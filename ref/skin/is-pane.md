## is-pane parameter (skin)

<!-- -->
**Applies to:**
+   [Main](/ref/%7Bskin%7D/control/main.md) <!-- -->
**Format:**
+   true/false
<!-- -->
**Default value:**
+   false


True if this is a pane that will be used in other container
controls, instead of an independent window. Read-only.
