## text parameter (skin)


**Applies to:**
+   [Button](/ref/skin/control/button.md) 
+   [Input](/ref/skin/control/input.md) 
+   [Label](/ref/skin/control/label.md) 
**Format:**
+   string


Text shown in this control. For Input controls this setting is
only available at runtime.
