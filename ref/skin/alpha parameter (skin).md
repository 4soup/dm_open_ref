## alpha parameter (skin)    
**See also:**    
:   [transparent-color parameter](/ref/%7Bskin%7D/param/transparent-color.md)    
<!-- -->    
**Applies to:**    
:   [Main](/ref/%7Bskin%7D/control/main.md) (window only)    
<!-- -->    
**Format:**    
:   number    
<!-- -->    
**Default value:**

Opacity of the window, from 0 (invisible) to 255 (opaque).
