## image-mode parameter (skin)

<!-- -->
**Applies to:**
+   [Label](/ref/%7Bskin%7D/control/label.md) 
+   [Main](/ref/%7Bskin%7D/control/main.md) <!-- -->

+   center
+   stretch
+   tile
