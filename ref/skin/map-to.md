## map-to parameter (skin) 
###### BYOND Version 511

<!-- -->
**Applies to:**
+   [Macro](/ref/skin/control/macro.md) 
<!-- -->
**Format:**
+   string


The [macro name](/ref/skin/macros.md) (e.g., \"SOUTH\") of a
key combo, Dpad, mouse button, etc. that this macro maps to.
