## is-minimized parameter (skin)

<!-- -->
**Applies to:**
+   [Main](/ref/skin/control/main.md) (window only)
<!-- -->
**Format:**
+   true/false
<!-- -->
**Default value:**
+   false


True if the window is minimized. 

If `is-fullscreen` is
true, this value represents the state of the window when `is-fullscreen`
is turned off again.
