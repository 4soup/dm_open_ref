## alpha parameter (skin)    
**See also:**    
:   [transparent-color parameter](/%7Bskin%7D/param/transparent-color)    
<!-- -->    
**Applies to:**    
:   [Main](/%7Bskin%7D/control/main) (window only)    
<!-- -->    
**Format:**    
:   number    
<!-- -->    
**Default value:**    
:   255    
Opacity of the window, from 0 (invisible) to 255 (opaque).
