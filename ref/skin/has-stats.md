##### BYOND Version 516

**Applies to:**
+   [Info](/ref/skin/control/info.md) 

**Format:**
+   true/false

**Default value:**
+   true

True if this info control contains the statpanels created via
[stat()](/ref/proc/stat.md) and [statpanel()](/ref/proc/statpanel.md)
Only one info control can have statpanels.
