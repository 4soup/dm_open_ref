## can-check parameter (skin)

<!-- -->
**Applies to:**
+   [Menu](/ref/skin/control/menu.md) <!-- -->
**Format:**
+   true/false
<!-- -->
**Default value:**
+   false


If true, this menu item is toggled like a checkbox or radio
button when clicked.
