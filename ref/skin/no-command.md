## no-command parameter (skin)

<!-- -->
**Applies to:**
+   [Input](/ref/%7Bskin%7D/control/input.md) <!-- -->
**Format:**
+   true/false
<!-- -->
**Default value:**
+   false


True if this input control is for typing only; hitting Enter
will not run a command.
