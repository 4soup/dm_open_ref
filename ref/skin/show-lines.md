## show-lines parameter (skin)
**See also:**
*   [line-color parameter](/ref/%7Bskin%7D/param/line-color.md) -m
<!-- -->
**Applies to:**
*   [Grid](/ref/%7Bskin%7D/control/grid.md) -m<!-- -->
**Possible values:**
*   none
*   horizontal
*   vertical
*   both
<!-- -->
**Default value:**
*   both


Determines which grid lines to display.
