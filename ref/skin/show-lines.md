## show-lines parameter (skin)

<!-- -->
**Applies to:**
+   [Grid](/ref/%7Bskin%7D/control/grid.md) <!-- -->
**Possible values:**
+   none
+   horizontal
+   vertical
+   both
<!-- -->
**Default value:**
+   both


Determines which grid lines to display.

> **See also:**
> +   [line-color parameter](/ref/%7Bskin%7D/param/line-color.md)
