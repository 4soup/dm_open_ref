##### BYOND Version 516

**Applies to:**
+   [Info](/ref/skin/control/info.md) 

**Format:**
+   true/false

**Default value:**
+   true

True if this info control contains the verbs used in the game.
Currently only one info control can have verbs.

> **See also:**
> +   [has-stats parameter](/ref/skin/param/has-stats.md)
