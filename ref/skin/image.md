## image parameter (skin)    
**See also:**    
:   [image-mode parameter](/ref/%7Bskin%7D/param/image-mode/image-mode.md)    
:   [keep-aspect parameter](/ref/%7Bskin%7D/param/keep-aspect/keep-aspect.md)    
<!-- -->    
**Applies to:**    
:   [Button](/ref/%7Bskin%7D/control/button/button.md)    
:   [Label](/ref/%7Bskin%7D/control/label/label.md)    
:   [Main](/ref/%7Bskin%7D/control/main/main.md)    
:   [Output](/ref/%7Bskin%7D/control/output/output.md)    
<!-- -->    
**Format:**    
:   \'*file*\'    
A background image to show in this control.    
In the Output control this image is always tiled.    
Note: Icons displayed in the output control will not show the background    
image underneath their transparent parts, but will instead show the    
background color.    
For Label and Main, use    
[image-mode](/ref/%7Bskin%7D/param/image-mode/image-mode.md){.code} to control how the    
image is displayed.
