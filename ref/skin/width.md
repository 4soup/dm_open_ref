## width parameter (skin)

<!-- -->
**Applies to:**
+   [Bar](/ref/skin/control/bar.md) <!-- -->
**Format:**
+   number
<!-- -->
**Default value:**
+   10


Width, in pixels, of the bar or slider. A value of 0 uses all
available width.

> **See also:**
> +   [dir parameter](/ref/skin/param/dir.md) 
> +   [is-slider parameter](/ref/skin/param/is-slider.md)
