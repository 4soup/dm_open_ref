## right-click parameter (skin)

<!-- -->
**Applies to:**
+   [All](/ref/%7Bskin%7D/control.md) <!-- -->
**Format:**
+   true/false
<!-- -->
**Default value:**
+   false


True if this control should allow right-clicks to behave like
any other click instead of opening up popup menus or similar special
behavior.
