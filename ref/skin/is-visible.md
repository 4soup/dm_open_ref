## is-visible parameter (skin)    
**See also:**    
:   [is-disabled parameter](ref/%7Bskin%7D/param/is-disabled)    
<!-- -->    
**Applies to:**    
:   [All](ref/%7Bskin%7D/control)    
<!-- -->    
**Format:**    
:   true/false    
<!-- -->    
**Default value:**    
:   true    
True if this control can be seen. The main window should usually be made    
visible.
