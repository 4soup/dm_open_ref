## right, bottom parameters (skin)

<!-- -->
**Applies to:**
+   [Child](/ref/skin/control/child.md) 
<!-- -->
**Format:**
+   string
+   none
<!-- -->
**Default value:**
+   none


The [id](/ref/skin/param/id.md) of the right/bottom pane
in this control. The parameter names `top` and `bottom` can be used
interchangeably.
