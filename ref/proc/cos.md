## cos proc
**See also:**
*   [arccos proc](/ref/proc/arccos.md) 
*   [sin proc](/ref/proc/sin.md) 
*   [tan proc](/ref/proc/tan.md) 
*   [turn proc](/ref/proc/turn.md) <!-- -->
**Format:**
*   cos(X)
<!-- -->
**Returns:**
*   The cosine of X, where X is in degrees.
### Example:

```
 mob/verb/test() usr \<\< cos(0) // 1 usr \<\< cos(45) //
0.707\... usr \<\< cos(90) // 0 
```
