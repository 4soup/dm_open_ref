## isicon proc

**Format:**
+   isicon(Icon)
<!-- -->
**Args:**
+   Icon: the value to test
<!-- -->
**Returns:**
+   1 if the value is a valid icon; 0 otherwise.


This returns a true value when given an icon. Both `/icon`
memory objects and icon files stored in the resource cache qualify.
