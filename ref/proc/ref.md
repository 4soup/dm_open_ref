## ref proc
**See also:**
+   [\\ref text macro](/ref/DM/text/macros/ref.md) 
+   [locate proc](/ref/proc/locate.md) <!-- -->
**Format:**
+   ref(Object)
<!-- -->
**Args:**
+   Object: The object to get a reference for
<!-- -->
**Returns:**
+   A string containing a reference ID.


This is a shortcut for `"\ref[Object]"`.
