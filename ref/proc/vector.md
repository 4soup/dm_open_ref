## vector proc 
###### BYOND Version 516

**Format:**
+   vector(x,y)
+   vector(x,y,z)
+   vector(List)
+   vector(Vector)

**Returns:**
+   A new vector.

**Args:**
+   x,y,z: Components of the new vector.
+   List: A numeric list to copy as a vector.
+   Vector: A vector to copy.


Creates a vector.
