## arcsin proc

**Format:**
+   arcsin(X)
<!-- -->
**Returns:**
+   The inverse sine of X in degrees.
### Example:

```
 mob/verb/test() usr \<\< arcsin(0) // 0 usr \<\<
arcsin(0.707107) // 45 usr \<\< arcsin(1) // 90 
```
