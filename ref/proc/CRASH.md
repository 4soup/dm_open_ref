## CRASH proc    
**See also:**    
:   [ASSERT proc](/proc/ASSERT)    
:   [DEBUG definition](/DM/preprocessor/define/DEBUG)    
<!-- -->    
**Format:**    
:   CRASH(msg)    
Crashes the current procedure, displaying the specified message and    
generating diagnostic debugging output, such as a stack dump.
