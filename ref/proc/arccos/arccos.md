  ## arccos proc
  **See also:**
  :   [arcsin proc](ref/proc/arcsin)
  :   [arctan proc](ref/proc/arctan)
  :   [cos proc](ref/proc/cos)
  :   [turn proc](ref/proc/turn)
  <!-- -->
  **Format:**
  :   arccos(X)
  <!-- -->
  **Returns:**
  :   The inverse cosine of X in degrees.
  ### Example:
  mob/verb/test() usr \<\< arccos(0) // 90 usr \<\< arccos(0.707107) // 45
  usr \<\< arccos(1) // 0
