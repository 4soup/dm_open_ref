## sin proc
**See also:**
*   [arcsin proc](/ref/proc/arcsin.md) -m
*   [cos proc](/ref/proc/cos.md) -m
*   [tan proc](/ref/proc/tan.md) -m
*   [turn proc](/ref/proc/turn.md) -m<!-- -->
**Format:**
*   sin(X)
<!-- -->
**Returns:**
*   The sine of X, where X is in degrees.
### Example:

```
 mob/verb/test() usr \<\< sin(0) // 0 usr \<\< sin(45) //
0.707\... usr \<\< sin(90) // 1 
```
