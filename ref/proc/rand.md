## rand proc    
**See also:**    
:   [pick proc](/proc/pick)    
:   [prob proc](/proc/prob)    
:   [rand_seed proc](/rand proc.md_seed)    
:   [roll proc](/proc/roll)    
<!-- -->    
**Format:**    
:   rand(L=0,H) or rand()    
<!-- -->    
**Returns:**    
:   A random integer between L and H inclusive; or a random number from    
    0 up to 1 if L and H are omitted.    
<!-- -->    
**Args:**    
:   L: A number for the lower-bound.    
:   H: A number for the upper-bound.
