## lowertext proc
**See also:**
*   [uppertext proc](/ref/proc/uppertext.md) -m<!-- -->
**Format:**
*   lowertext(T)
<!-- -->
**Returns:**
*   A lowercase text string.
<!-- -->
**Args:**
*   T* A text string.


Make all of the characters of T lowercase.
### Example:

```
 L = lowertext(\"HI THERE\") // = \"hi there\" 
```
