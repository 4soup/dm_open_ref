**Format:**
:   sin(X)
<!-- -->
**Returns:**
:   The sine of X, where X is in degrees.
### Example:
mob/verb/test() usr \<\< sin(0) // 0 usr \<\< sin(45) // 0.707\... usr
\<\< sin(90) // 1
