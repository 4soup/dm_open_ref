## arccos proc

**Format:**
+   arccos(X)
<!-- -->
**Returns:**
+   The inverse cosine of X in degrees.
### Example:

``` dm
 mob/verb/test() usr << arccos(0) // 90 usr <<
arccos(0.707107) // 45 usr << arccos(1) // 0 
```
