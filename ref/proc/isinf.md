## isinf proc 
###### BYOND Version 515

<!-- -->
**Format:**
+   isinf(n)
<!-- -->
**Args:**
+   n: A number
<!-- -->
**Returns:**
+   1 if this is an infinite numeric value, either positive or negative;
    0 otherwise
