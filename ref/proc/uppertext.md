## uppertext proc
**See also:**
*   [lowertext proc](/proc/lowertext)
<!-- -->
**Format:**
*   uppertext(T)
<!-- -->
**Returns:**
*   A capitalized text string.
<!-- -->
**Args:**
*   T* A text string.


Capitalize all of the characters of T.
### Example:

```
 U = uppertext(\"hi there\") // = \"HI THERE\" 
```
