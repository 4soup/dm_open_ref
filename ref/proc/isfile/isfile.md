## isfile proc
**See also:**
:   [cache](#/DM/cache)
:   [file proc](#/proc/file)
:   [isicon proc](#/proc/isicon)
<!-- -->
**Format:**
:   isfile(File)
<!-- -->
**Args:**
:   File: the value to test
This returns a true value when given a file. Both objects returned by
file() and files stored in the resource cache qualify.
