## == operator

**Format:**
+   A == B
<!-- -->
**Returns:**
+   1 if A and B are equal; 0 otherwise


Note that this is not the same as the assignment operator (=),
which sets A equal to B. 

To check if A and B are equivalent,
use the \~= operator.
