## \<\<= operator    
**See also:**    
:   [\>\>= operator](ref/operator/%3e%3e=)    
:   [\<\< shift operator](ref/operator/%3c%3c/shift)    
:   [operators](ref/operator)    
<!-- -->    
**Format:**    
:   A \<\<= B    
Set A equal to A \<\< B. It is shorthand for A = A \<\< B.
