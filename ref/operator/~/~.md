## \~ operator    
**See also:**    
:   [operators](ref/operator)    
<!-- -->    
**Format:**    
:   \~ A    
<!-- -->    
**Returns:**    
:   The binary \"not\" of A.    
A must be between 0 and 2\*\*24 - 1, giving an effective width of 24    
bits.    
If A is a /matrix datum, the result is a new matrix which is the inverse    
of A.
