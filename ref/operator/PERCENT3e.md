## \> operator    
**See also:**    
:   [\>= operator]/&gt; operator.md=    
:   [\< operator]/operator/%3c

<!-- -->    
**Format:**    
:   A \> B    
<!-- -->    
**Returns:**    
:   1 if A is greater than B; 0 otherwise.    
If A and B are text strings, a case sensitive comparison is performed    
(like sorttextEx()).
