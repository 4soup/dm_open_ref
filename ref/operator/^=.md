## \^= operator    
**See also:**    
:   [\^ operator](/operator/%5E)    
:   [operators](/operator)    
<!-- -->    
**Format:**    
:   A \^= B    
Set A equal to A \^ B. It is shorthand for A = A \^ B.    
If A and B are lists, any items that are found in both lists are removed    
from A, and then any items found only in B are added to A.
