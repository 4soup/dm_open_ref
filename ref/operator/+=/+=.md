## += operator    
**See also:**    
:   [\*= operator](/ref/operator/*=)    
:   [+ operator](/ref/operator/+)    
:   [-= operator](/ref/operator/-=)    
:   [/= operator](/ref/operator//=)

<!-- -->    
**Format:**    
:   A += B    
Set A equal to A + B. It is shorthand for A = A + B.    
If A is an /icon, /matrix, pixloc, or vector, it will be changed rather    
than creating a new value and re-assigning it to A.
