  ## \>\> input operator
  **See also:**
  :   [\>\> operator (savefile)](ref/savefile/operator/%3e%3e)
  :   [file proc](ref/proc/file)
  <!-- -->
  **Format:**
  :   F \>\> Var
  Cause input to be read from a file into a variable. The file may be a
  savefile or a file object corresponding to a text file.
