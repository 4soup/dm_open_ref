## \< operator

:   [== operator](/operator/==)

<!-- -->    
**Format:**    
:   A \< B    
<!-- -->    
**Returns:**    
:   1 if A is less than B; 0 otherwise.    
If A and B are text strings, a case sensitive comparison is performed    
(like sorttextEx()).
