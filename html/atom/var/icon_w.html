icon_w var (atom)

See also:
icon_z var (atom)
pixel_w var (atom)
icon var (atom)
Default value:
0

Defines the anchor point for this icon, from the bottom left. Higher values of icon_w will offset this icon to the left; lower values offset it to the right. It acts like a negative pixel_w except it's not inherited by any overlays, images, or visual contents. It applies ONLY to this icon.

This var is meant to replace the bound_x var, which was applied very inconsistently in old versions.
