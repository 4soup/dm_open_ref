trunc proc

See also:
fract proc
floor proc
ceil proc
round proc
Format:
trunc(A)
Returns:
truncated A
Args:
A: A number, pixloc, or vector.

Returns the integer part of the number A. That is, this rounds toward 0 to an integer.

Example:

usr << trunc(1.45) // outputs 1 usr << trunc(-1.45) // outputs -1
